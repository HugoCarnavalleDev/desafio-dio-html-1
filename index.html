História da Computação

Voltar

Gerações de Computadores

1ª Geração (1940-1956): Nesta época, os computadores eram enormes, ocupando salas inteiras, e usavam válvulas e painéis de controle para processar dados. Um exemplo famoso é o MARK I, que foi um dos primeiros computadores digitais.

2ª Geração (1956-1963): Os computadores da segunda geração eram menores, mais rápidos e mais confiáveis do que os da geração anterior. Eles usavam transistores em vez de válvulas, o que os tornava mais eficientes energeticamente. Os sistemas operacionais começaram a ser desenvolvidos nesta época.

3ª Geração (1964-1971): Os computadores da terceira geração usavam circuitos integrados, que eram chips contendo vários transistores. Isso os tornava ainda menores, mais rápidos e mais baratos de produzir. Os computadores podiam ser usados por várias pessoas ao mesmo tempo, graças ao desenvolvimento dos sistemas operacionais multiusuários.

4ª Geração (1971-até os dias atuais): Com o surgimento dos microprocessadores, os computadores se tornaram ainda menores, mais poderosos e mais acessíveis. Os microprocessadores eram chips que continham todos os componentes principais de um computador em um único circuito integrado, o que permitia que os computadores pessoais se tornassem uma realidade.

Essas quatro gerações marcaram avanços significativos na evolução dos computadores, levando a mudanças fundamentais em seu tamanho, capacidade e acessibilidade. Segundo o site Toda matéria:

"nós pertencemos à quarta geração dos computadores, o que revela uma evolução incrível nos sistemas de informação. Um ponto de destaque é que a evolução dos computadores ocorria de maneira mais lenta. Com o desenvolvimento da sociedade, meios de comunicação e a divulgação científica podemos ver a evolução dessas máquinas em dias ou meses. Alguns estudiosos preferem acrescentar a “Quinta Geração de Computadores” com o aparecimento dos supercomputadores, utilizados por grandes corporações como a NASA. Nessa geração, é possível avaliar a evolução da tecnologia multimídia, da robótica e da internet".

Início da Internet

A Internet começou nos anos 1960 como um projeto do governo dos Estados Unidos para criar uma rede de comunicação que fosse resistente a falhas. Chamada de ARPANET, ela conectava inicialmente universidades e instituições de pesquisa.

Nos anos 1980, a ARPANET evoluiu para o que conhecemos como Internet, conectando mais computadores e se espalhando pelo mundo acadêmico.

Na década de 1990, a Internet se tornou acessível ao público em geral com o surgimento da WWW* e dos navegadores, como o Netscape. Isso levou a um crescimento explosivo da Internet, tornando-se uma parte essencial da vida moderna.

Desde então, a Internet continuou a crescer e evoluir, se tornando um recurso fundamental para comunicação, pesquisa, entretenimento e trabalho em todo o mundo.

*A ideia da WWW começou na Suíça em 1980. Tim Berners-Lee, um britânico, teve a ideia. Ele usou um computador NeXTcube como o primeiro servidor web e para criar o primeiro navegador, chamado WorldWideWeb, em 1990.

O que são Clients

Voltar

O lado do cliente na web se refere aos elementos de um aplicativo web que são processados e exibidos no dispositivo do usuário final. Isso inclui o conteúdo visualizado pelo usuário e qualquer interação que o aplicativo execute no navegador do usuário. Linguagens como HTML, CSS e JavaScript são interpretadas pelo navegador no lado do cliente. Muitos aplicativos web modernos movem parte da lógica de negócios para o lado do cliente, permitindo interações mais dinâmicas e responsivas, como no site da Netflix. O termo "lado do cliente" é semelhante, mas não idêntico, ao front-end, que se refere aos processos relacionados à interação direta com o usuário final.

O que são Servers

Voltar

O lado do servidor, é a parte de um site que não vemos diretamente, mas que faz ele funcionar. Ele processa as coisas que pedimos, como abrir uma página ou enviar uma mensagem. Por exemplo, quando você entra em um site, o servidor pega o que você pediu, como uma página de notícias, e a envia de volta para você ver no seu navegador. Uma das vantagens do server-side é a segurança, pois ele guarda informações importantes, como senhas, de forma protegida. Além disso, ele ajuda a controlar como o site funciona e a lidar com muitos usuários ao mesmo tempo, tornando possível criar sites complexos e que podem crescer conforme necessário.

Linguagens de Programação

Voltar

As linguagens de programação são criadas para permitir que os programadores instruam os computadores. Enquanto as linguagens naturais, como o português inglês, são usadas para comunicação entre pessoas, linguagens de programação são artificiais e servem para interagir com máquinas. Quando os programadores escrevem um programa, eles estão dando uma sequência de instruções que dizem ao computador o que fazer. Existem muitas linguagens de programação diferentes, e algumas são mais populares porque são mais fáceis de entender e usar. Essas linguagens são como um tipo de código que os computadores conseguem entender. Elas são traduzidas para um código especial chamado código binário, que o computador consegue entender e executar. Assim, as linguagens de programação permitem que os computadores cumpram as instruções dadas pelos programadores, tornando possível criar softwares e sistemas que usamos todos os dias.

Falando sobre Tags

Voltar

Tags no HTML são elementos usados para estruturar e formatar o conteúdo de uma página web. Elas são compostas por um nome entre os símbolos de maior e menor, como <tag>, e são normalmente usadas em pares, uma tag de abertura e uma tag de fechamento, como <tag>conteúdo</tag>.

Exemplos de tags:

<i>: é usada para colocar parte do texto em itálico, dando-lhe uma voz ou humor diferentes. É comumente usada para destacar termos técnicos, frases em outro idioma, pensamentos, nomes, entre outros.

<strong>: é usada para destacar texto importante, que é mostrado em negrito.

<input>: cria um espaço onde as pessoas podem digitar informações. É um dos elementos mais importantes em formulários e pode parecer de diferentes formas, dependendo do tipo de informação que se espera receber.

<img>: é usada para adicionar uma imagem. Em vez de serem inseridas diretamente na página, as imagens são vinculadas a ela. Ela cria um espaço onde a imagem será exibida.

Atributos das Tags

Voltar

Os tributos no HTML são características adicionais que podem ser somadas às tags para fornecer informações extras sobre como um elemento deve ser exibido ou se comportar.

Exemplos de atributos globais:

id: especifica um identificador único para um elemento HTML, que deve ser único no documento. Esse atributo é comumente usado para manipular o elemento usando JavaScript.

style: define um estilo específico para um elemento em uma página da web. Esse estilo vai substituir qualquer estilo definido de forma global, Esse atributo é comumente usado para manipular o elemento usando CSS.

class: indica uma ou mais classes para um elemento HTML. Geralmente, é usado para aplicar estilos a partir do CSS, mas também pode ser usado pelo JavaScript para modificar elementos com essa classe.

Exemplos de atributos específicos:

type: especifica o tipo de elemento a ser exibido entro do campo do formulário, podendo ser númerico, texto, cor entre outros.

src: especifica o local (URL) da imagem externa.

width: especifica a largura de um elemento, em pixels.

Textos

Voltar

Os cabeçalhos <h1> a <h6> são usados para organizar e destacar títulos em uma página web. Os motores de busca usam esses cabeçalhos para entender a estrutura e o conteúdo da página, enquanto os usuários geralmente os utilizam para navegar rapidamente pelo conteúdo. É importante usar os cabeçalhos na ordem correta, começando pelo <h1> para os títulos principais, seguido do <h2> para subtítulos, e assim por diante.

Exemplos de tags para formatação de texto:

<p>: é usado para criar um parágrafo. Sempre que você o utiliza, o texto começa em uma nova linha, e o navegador adiciona automaticamente um espaço em branco antes e depois do parágrafo.

<hr>: é usada para criar uma linha horizontal que separa conteúdos em uma página HTML. É como uma linha divisória que ajuda a organizar e diferenciar partes do texto.

<u>: é usada para destacar um texto de maneira diferente, geralmente colocando uma linha embaixo dele. Pode ser usado para indicar palavras escritas erradas, por exemplo. É possível mudar o estilo dessa linha usando CSS.

<mark>: é usada para destacar um texto, como quando você marca um trecho importante em um documento.

Listas Ordenadas e Não Ordenadas

Voltar

Listas em HTML permitem que desenvolvedores agrupem um conjunto de itens relacionados em listas.

Exemplos de tags de lista:

<ol>: define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética. A tag <li> é usada para definir cada item da lista.

<ul>: é usada para fazer listas sem ordem (com marcadores). Para criar essas listas, use a tag <ul> junto com a tag <li>.

Links

Voltar

A tag <a> é usada para criar um link para outra página. O atributo mais importante dela é o href, que indica para onde o link deve apontar. Por padrão um link não visitado é azul e sublinhado.

Já o atributo target indica onde abrir o documento vinculado. Quando usamos o valor "-blank", o documento linkado é aberto em uma nova janela ou aba. Já o valor "-self" abre o documento linkado no mesmo quadro em que foi clicado, que é a configuração padrão.
